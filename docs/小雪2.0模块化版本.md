# 小雪工作台 2.0.0｜模块化版

> 发布日期：2026-07-18
> 项目：小雪电竞交易助理（LOL 为当前主业）

## 这次版本代表什么

2.0.0 是小雪工作台的大版本重构：把原来集中在少数大文件里的能力拆成可独立运行、独立检查、互不拖垮的模块。

## 统一后的结构

- 表现层：页面和交互，只负责展示与用户操作。
- 业务逻辑层：各模块自己的判断、编排和状态。
- 数据层：API、数据库、Wiki/TK 等资料读取与写回边界。
- 模块接口：模块之间只通过公开接口和事件通信，不直接互相调用内部实现。

## 当前版本边界

- 工作台主窗口：队伍资料、当前赛事、TK 资料库。
- 临场记录：保留为隐藏辅助模块，只读/手写记录，不自动交易。
- 日报：模块已保留，但按钧钧决定暂时手动暂停，待剩余模块修好后再恢复。
- 纯十英雄阵容分析、交易系统和日报能力继续使用各自独立版本号，不强行改成 2.0.0。
